# KFC Landing Page – Improved Version

This project is a refined version of a KFC landing page that I forked and enhanced with a focus on better user experience.

## Improvements Made

### 1. Background Gradient Enhancement
* Updated the **direction of the gradient background** to create a more modern look.
### 2. Added Animations
* Introduced **animations across different elements** of the page.
* These animations improve interactivity
### 3. Header Menu Optimization
* Improved the **spacing and font size** of the navigation menu.
* Fixed the issue where menu items appeared too small and too close to the hero image.
### 4. Text Spacing & Typography Improvements
* Enhanced the **spacing and font sizes** of the main content in the middle section.
* This makes the text easier to read
### The goal of these updates was to:
* Improve **UI/UX design**
* Enhance **readability**
* Add **modern visual touches** through animation and spacing
